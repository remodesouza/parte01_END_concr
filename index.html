Como Navegar nas Apresentações

  • Use as setas do teclado (→ e ←) para avançar ou voltar nos slides.
  • Nas listas com tópicos, pressione a seta para a direita (→) para revelar cada item passo a passo.
  • Para slides com carrossel de imagens, clique nas setas do carrossel (◀ e ▶) para trocar de imagem.
  • Você também pode usar o mouse ou toque (em dispositivos móveis) para navegar.
